# Cách chạy ứng dụng web
1- Sửa file cấu hình firebase trong src/firebaseConfig.jsx 
```js
const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
  measurementId: "",
};
```
2- chạy lệnh tải các gói cần thiết `npm install`
3- Chạy ứng dụng `npm run dev`
